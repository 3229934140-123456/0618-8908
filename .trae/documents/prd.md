## 1. 产品概述

智能家居设备集中控制面板是一个基于Web端的统一管理平台，让用户将各品牌智能设备通过官方API接入，实现跨品牌设备的统一管控、自动化场景编排、异常事件推送与能耗统计，告别多App切换的碎片化体验。

- 目标用户：拥有多品牌智能设备的家庭用户，追求一站式便捷管理
- 核心价值：消除品牌壁垒，统一控制入口，智能化联动，家庭安全守护

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 家庭管理员 | 邮箱注册 | 全部设备管理、成员管理、场景创建、权限分配 |
| 家庭成员 | 管理员邀请加入 | 被授权范围内的设备查看与控制 |

### 2.2 功能模块

1. **首页仪表盘**：设备状态总览、快捷场景触发、异常告警提示、能耗速览
2. **设备管理页**：设备列表/分组、设备详情与控制、品牌API接入配置
3. **自动化场景页**：场景列表、场景创建/编辑、条件触发器配置
4. **操作记录页**：历史操作时间线、按设备/成员/时间筛选
5. **能耗统计页**：月度用电量图表、设备能耗排行、趋势分析
6. **家庭成员页**：成员列表、权限设置、邀请成员

## 3. 核心流程

### 3.1 设备接入流程
用户进入设备管理页 → 点击添加设备 → 选择品牌 → 输入API密钥/授权 → 系统获取设备列表 → 用户选择要接入的设备 → 设备出现在面板中

### 3.2 场景创建流程
用户进入自动化场景页 → 点击新建场景 → 命名场景 → 设置触发条件（定时/地理围栏/设备状态） → 添加执行动作（控制设备参数） → 保存并启用

### 3.3 异常告警流程
设备检测到异常 → 系统判断告警规则 → 生成告警通知 → 推送到首页仪表盘和通知中心 → 用户查看并处理

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深空黑 (#0B0D17) + 暗夜蓝 (#121629) 作为主背景，营造沉浸式科技感
- **强调色**：琥珀金 (#F5A623) 作为主要强调色，温暖而不失高级；冰蓝 (#4FC3F7) 作为辅助强调色
- **功能色**：成功绿 (#66BB6A)、警告橙 (#FFA726)、危险红 (#EF5350)、信息蓝 (#42A5F5)
- **按钮风格**：圆角微凸起，悬浮时发光效果，点击时下沉反馈
- **字体**：标题使用 Outfit（几何感科技风），正文使用 DM Sans（高可读性）
- **布局风格**：左侧固定导航栏 + 主内容区卡片式布局，卡片采用毛玻璃质感
- **图标风格**：线性图标配合微妙发光效果，统一使用 Lucide Icons
- **动效**：页面切换淡入淡出，卡片悬浮轻微上浮+阴影加深，设备状态变化脉冲动画

### 4.3 响应式设计

- 桌面端优先（1920px+），兼顾笔记本（1366px+）和平板（768px+）
- 导航栏在768px以下转为底部标签栏
- 卡片网格自适应：桌面4列 → 平板2列 → 手机1列
- 触控优化：按钮最小44px点击区域，滑块控件适配触摸操作

### 4.4 3D场景指引

不适用，本项目不涉及3D场景。
